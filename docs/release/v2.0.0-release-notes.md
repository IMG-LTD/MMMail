# MMMail v2.0.0 Release Notes

## Highlights
- `MMMail v2.0.0` promotes the frozen v2 route, contract, and milestone architecture as the new execution baseline.
- Extraction-ready backend modules and shared `frontend-v2` contracts are included in the release scope.
- Preview modules remain available under `/labs/:moduleKey` while staying outside the GA top-level product baseline.

## Validation
- `bash scripts/validate-local.sh`
- `frontend-v2` participates in both local and CI release gates for `v2.0.0`.

## Upgrade Notes
- Treat `release/2.0.0` as the integration branch until the release is merged into `main`.
- Use this release note as the source for the GitHub Release publication step.
