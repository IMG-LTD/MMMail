推荐流程 / Suggested Workflow

  1. 先验证健康检查：curl -sf <api-base>/actuator/health
  2. 再打开 Swagger UI，确认当前部署实际暴露的路由、参数与响应结构。
  3. 需要与工具链对接时，直接读取 OpenAPI JSON，并把结果和当前运行时配置一起比对。
  4. 如遇 origin、鉴权或 CORS 问题，优先回查 `NUXT_PUBLIC_API_BASE` 与后端安全配置，而不是猜测前端地址。
